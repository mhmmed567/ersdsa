# **App Name**: Warm Hearth

## Core Features:

- Menu & Product Browsing: Users can explore categorized products, view details, prices, and high-quality images, supporting a sophisticated browsing experience.
- Shopping Cart Management: Customers can intuitively add items to a cart, easily adjust quantities, and remove unwanted products before finalizing an order.
- Guest Checkout & Order Placement: Streamlined checkout process allowing customers to place orders by providing essential contact and pickup/delivery details without needing an account.
- Order Management Dashboard (Staff): A dedicated dashboard for staff to view incoming orders, track their status, update fulfillment progress, and manage order details efficiently.
- AI-Powered Order Summary Tool: For staff, a generative AI tool to instantly provide concise summaries of customer orders, including any special requests or notes, streamlining order fulfillment.
- User Role Selection: A clear entry point for users to select their role (e.g., customer or employee), guiding them to the appropriate interface and functionality.

## Style Guidelines:

- Primary color (for text, icons, important details): Dark Brown (#2a1810) to convey seriousness and luxury.
- Accent color (for titles, buttons, interactive links): Copper Orange (#c07446) to attract attention and add warmth.
- Background color: Light Beige (#faf7f2) as a calm, eye-friendly base that highlights other colors.
- Success color: Dark Green (#0c4b40) for confirmation messages, such as order success.
- Highlight contrast color: White (#ffffff) for backgrounds of elements like main buttons to enhance visibility.
- Body and paragraph text: 'Inter' (sans-serif) for general content and 'Cairo' (sans-serif) specifically for clear and comfortable Arabic reading. Note: currently only Google Fonts are supported.
- Headlines and prominent text: 'Outfit' (sans-serif) for a modern and attractive style. Note: currently only Google Fonts are supported.
- Classical and detail text: 'Cormorant Garamond' (serif) for elegant, classic touches in specific areas. Note: currently only Google Fonts are supported.
- Utilize elegant and simple iconography that aligns with the luxurious and earthy design theme, maintaining a clean visual language.
- Implement clean, organized layouts with generous white space across all pages, ensuring readability and a premium feel, optimized for responsiveness.
- Incorporate subtle and smooth transitions for interactive elements such as buttons, selections, and content changes, contributing to a polished user experience.
